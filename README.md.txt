# Attention Mapping Dashboard

This project monitors user attention by tracking keyboard and mouse activity.  
It detects active window changes, alerts users when distracting apps are opened, and visualizes engagement levels using heatmaps.

## Features
- Tracks keyboard activity
- Tracks mouse activity
- Detects active window changes
- Alerts when distracting apps are opened
- Generates attention heatmap
- Shows time spent on different applications

## Technologies Used
- Python
- Tkinter
- Matplotlib
- pynput
- pygetwindow

## How to Run

1. Install dependencies
pip install pynput pygetwindow matplotlib

2. Run the program
python main.py
